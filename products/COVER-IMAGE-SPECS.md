# Gumroad Cover-Image Design Specs
**Format**: 1280x720px (16:9) oder 1200x900px (4:3) — Gumroad empfiehlt 1280x720
**Tool**: Canva Free (canva.com) — kein Designer nötig

---

## Einheitliches Design-System

### Farben (Konsistentes Branding)
- **Hintergrund**: `#0F172A` (Dark Navy)
- **Akzent**: `#6366F1` (Indigo/Lila)
- **Text primary**: `#F8FAFC` (White)
- **Text secondary**: `#94A3B8` (Slate)
- **Badge/Highlight**: `#10B981` (Emerald Green)

### Schriften
- **Titel**: Inter Bold (in Canva verfügbar)
- **Untertitel**: Inter Regular
- **Badge**: Inter SemiBold

### Template-Struktur (alle 5 Produkte)
```
┌─────────────────────────────────────────┐
│  [Logo oben links]     [Badge oben rechts]│
│                                           │
│  [Emoji/Icon groß - Mitte]               │
│                                           │
│  PRODUKT TITEL                           │
│  (groß, weiß, Bold)                      │
│                                           │
│  Kurzbeschreibung (grau, kleiner)        │
│                                           │
│  [3 Features als Chips] [Preis Badge]    │
└─────────────────────────────────────────┘
```

2. Farben auf Design-System oben anpassen
3. Icons aus Canva-Bibliothek (kostenlos)
4. Text einfügen nach obigen Specs
5. Als PNG 1280x720 exportieren

- Ich erstelle HTML-Templates die ihr als Screenshot speichern könnt
- Browser → 1280x720 Fenster → Screenshot
